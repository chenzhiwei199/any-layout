## any-layout

拖拽式的布局组件，可以通过拖拽的方式，快速生成嵌套布局、grid布局、绝对布局。

### 安装

`tnpm install any-layout`

### 维护人员

czw  czwcode@gmail.com

### 特点

- 同时支持嵌套布局、grid布局、绝对布局
- 通过一套schema描述三种布局，并具备较强的扩展性，用户可以通过实现布局组件的方式，扩展各种布局。

### 兼容性说明

表格用到了大量较新的浏览器 API，目前阶段只兼容较新版本的 Chrome/Firefox/Safari
